# Python Simple Car Game

This is a very simple and fun top-down car dodging game written in Python, meant for passing time!

## 🚦 Requirements

The game requires Python and a graphics library called **Pygame**. 
If you don't have Pygame installed, you have to install it first.

### 1. Install Pygame
Open your Terminal, Command Prompt, or PowerShell, and run the following command to download and install Pygame:

```bash
pip install pygame
```

### 2. How to Run the Game
Navigate to the directory where the game is located, and execute the python script:

```bash
python car_game.py
```
*(If the command `python` does not work, try `python3` instead)*

## 🎮 How to Play

1. Use the **Left Arrow (`←`)** and **Right Arrow (`→`)** keys on your keyboard to navigate the Green Car.
2. Avoid crashing into the incoming **Red obstacles**.
3. Every time an obstacle goes off-screen, your score increases, and the game will slowly get **faster and harder!**
4. If you hit an obstacle, it's Game Over!
5. After losing, press **`R`** to restart instantly, or press **`Q`** to quit.

Enjoy your game!
